# 医院后台管理系统 java921

## 项目概述

医院后台管理系统是基于 Django 框架开发的一套便捷、高效的业务支持系统。系统支持二次开发，旨在帮助医院实现信息化管理，提高工作效率。

## 技术栈

- 后端框架：Django
- 数据库：SQLite
- 编程语言：Python
- 前端技术：HTML

## 功能模块

### 系统角色

- 管理员页面

### 权限控制

- 对不同角色用户进行权限控制

### 功能页面

- 登陆页面
- 管理人员/添加组页面
- 医生管理页面
- 患者管理页面
- 病房管理页面
- 科室管理页面
- 药品管理页面

### 基础操作

- 增删改查功能

## 运行环境

- Python 环境搭建
- SQLite 数据库支持
